# Seedlings Database Project 

Цей проєкт містить базу даних розсади овочів, фруктів та екзотичних рослин для вебсайту. 
Виконано в рамках вивчення обробки даних на Python.

## Про дані
Файл `база даних.csv` містить 25 найменувань, розділених на категорії:
*   **Овочі** (Томати, Огірки, Капуста тощо)[cite: 1]
*   **Ягоди та Фрукти** (Полуниця, Лохина, Виноград)[cite: 1]
*   **Баштанні** (Кавун, Диня)[cite: 1]
*   **Екзотика** (Лимон, Інжир, Гранат)[cite: 1]

Кожен запис має параметри: назва, категорія, ціна, складність догляду та режим поливу[cite: 1].

## Як це працює
Код у репозиторії зчитує CSV-файл за допомогою бібліотеки `pandas` та конвертує його у формат **list of dictionaries**. 
Це дозволяє легко використовувати дані для:
1. Виводу карток товарів на сайті.
2. Фільтрації за ціною або категорією.
3. Аналізу асортименту.
